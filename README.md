# טק-בלוג – אתר Jekyll בעברית

אתר מודרני בנושאי טכנולוגיה, פיתוח תוכנה, בינה מלאכותית וסייבר.

## הרצה מקומית

```bash
bundle install
bundle exec jekyll serve
```

## פריסה ב-GitHub Pages

1. העלו את הפרויקט ל-GitHub
2. הפעילו GitHub Pages בהגדרות הריפוזיטורי
3. בחרו את הענף `main` כמקור

## מבנה

- `_posts/` – פוסטים קצרים
- `_articles/` – מאמרים ארוכים
- `_projects/` – פרויקטים
- `_layouts/` – תבניות עמוד
- `_includes/` – רכיבים משותפים
- `assets/css/` – עיצוב
- `pages/` – דפים סטטיים
